bootstrap响应式按钮菜单

2018-12-07

日期: 2018-12-07 19:55:14

为了更好的适配移动端，采用bootstrap框架对css进行简化操作，并且得到了很好的效果。

这个响应式菜单按钮在个人微博中非常常见，包括自己的个人微博。

今天在学习中进行了一个实现。

1.添加固定的meat

<meta charset="UTF-8" name="viewport" content="width=device-width, initial-scale=1.0">

2.引入bootstrap.css文件和bootstrap.js及jQuery.js文件

<link rel="stylesheet" href="css/bootstrap.min.css">
<script src="js/jQuery-1.9.1-min.js"></script>
<script src="js/bootstrap.min.js"></script>

3.主要布置

<!--常见响应式导航菜单-->
<!--三个div，第一个大div盒子-->
<div class="nav navbar-default navbar-fixed-top">
<!--第二个div，支持移动设备-->
<div class="navbar-header">
    <a class="navbar-brand">前端技术</a><!--导航头信息，保证显示-->
    <!--button的响应式样式,屏幕宽度小于768px时，div.navbar-responsive-collapse容器里的内容都会隐藏，显示icon-bar图标，当点击icon-bar图标时，再展开。屏幕大于768px时，默认显示。-->
    <button class="navbar-toggle" type="button" data-toggle="collapse" data-target=".navbar-responsive-collapse">
        <span class="icon-bar"></span>
        <span class="icon-bar"></span>
        <span class="icon-bar"></span>
    </button>
</div>
<div class="collapse navbar-collapse navbar-responsive-collapse">
    <ul class="nav navbar-nav">
        <li class="active"><a href="##">Home</a></li>
        <li><a href="##">CSS3</a></li>
        <!--添加一个二级菜单下拉样式-->
        <li class="dropdown" id="box">
            <a href="##" class="dropdown-toggle" data-toggle="dropdown">Sass<span class="caret"></span></a>
            <ul class="dropdown-menu">
                <li><a href="##">001</a></li>
                <li><a href="##">002</a></li>
                <li><a href="##">003</a></li>
            </ul>
        </li>
        <li><a href="##">jQuery</a></li>
        <li><a href="##">Responsive</a></li>
    </ul>
   </div>
</div>
