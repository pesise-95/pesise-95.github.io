JavaScript添加变量使事件进程相对独立

2018-11-30

日期: 2018-11-29 20:10:22

今天在做楼层导航时， 发现点击事件与窗口滚动事件“粘粘在一起”。
想了很久，于是引用进变量监听点击事件发生的状态来解决。

1.引用变量监听

var isClick = false;

默认情况下为false，即表示点击事件未发生

2.在点击事件中

2.1 在点击事件开始时设置状态

isClick = true

2.2 在点击事件动画完成后的同时清除定时器并且设置状态

if (target === end) {
   clearInterval(timer);
   isClick = false;//动画结束时为false
 }

3.在窗口滚动事件开始前判断isClick的状态

if (!isClick){}//当动画结束后再判断导航栏与内容的位置关系

以上，解决了事件之间“粘连”的问题。
